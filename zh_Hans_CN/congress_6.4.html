Civ VI BBG 6.4 世界议会


军备控制

所有文明的大规模杀伤性武器储量将与目标文明储量匹配。

目标文明将失去所有大规模杀伤性武器。

Earliest Era: 现代


边境控制条约

该文明的单元格购买费用降低50%。

该文明的边界将无法通过 文化值增长。

Latest Era: 现代


森林砍伐条约

禁止砍伐或清除所选地貌类型。

清除所选地貌类型产出的 金币等同于 生产力与 食物。

Earliest Era: 工业时代

Latest Era: 信息时代


间谍协定

间谍执行所选行动时产生的效果提升2级。

选择的行动不可用。

Earliest Era: 文艺复兴时期

Latest Era: 原子能时代


全球能源条约

禁止建造此类型建筑。

建造此类型建筑所需生产力减少50%。

Earliest Era: 文艺复兴时期


文化遗产组织

该类型巨作的 旅游业绩翻倍。

该类型巨作将不提供 旅游业绩。

Earliest Era: 工业时代


奢侈品政策

所选奢侈品资源将重复提供 宜居度。

所选奢侈品资源将无法提供 宜居度。


雇佣兵公司

下次世界议会召开前，使用此类型货币建造或购买军事单位时花费+50%。

下次世界议会召开前，使用此类型货币建造或购买军事单位时花费-25%。


移民条约

该文明的城市 人口增长+20%，但每回合将-5忠诚度。

该文明的城市每回合+5忠诚度，但 人口增长将-20%。

Earliest Era: 中世纪


军事顾问

拥有所选升级类型的单位+3战斗力。

拥有所选升级类型的单位-3战斗力。

Latest Era: 原子能时代


赞助

获得此类型 伟人两倍的点数。

无法获得此类型 伟人的点数。

Latest Era: 现代


公共工程计划

建造该项目时+100% 生产力。

建造该项目时-50% 生产力。

Earliest Era: 工业时代

Latest Era: 信息时代


主权

通往该类型城邦的 贸易路线建立后城邦的此类产出+100%。

此类型的城邦将无法提供自身特色宗主国加成。

Latest Era: 现代


贸易政策

通往所选国家的 贸易路线可为其发起者提供+4 金币。所选国家获得+1 贸易路线容量。

通往或源自所选文明的国际 贸易路线-4金币。（可变为负值）


城市发展条约

建造该区域中的建筑时+100% 生产力。

该区域中无法修建建筑。

Latest Era: 现代


世界意识形态

该政体类型获得一个通配符政策槽位。

该政体类型失去一个通配符政策槽位。

Earliest Era: 中世纪


世界宗教

为此 宗教的所有宗教单位+10 宗教战斗力。

所有文明可对所选宗教的单位进行“异教徒审判”，审判后可获得25点 外交支持。

Latest Era: 工业时代
