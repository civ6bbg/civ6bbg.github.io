Civ VI BBG 6.2 Congrès mondial


Contrôle des armes

Les armes de destruction massive de tous les joueurs sont réglées sur le niveau de celles du joueur ciblé.

Le joueur ciblé perd toutes ses armes de destruction massive.

Earliest Era: Ère moderne


Traité de contrôle des frontières

L'achat par ce joueur de nouvelles cases est réduit de 50%.

Les frontières de ce joueur ne s'élargissent pas avec la Culture.

Latest Era: Ère moderne


Traité de déforestation

Empêche de supprimer des caractéristiques du type correspondant.

Supprimer des caractéristiques du type correspondant génère une quantité d'or équivalente à la Production et à la Nourriture générées.

Earliest Era: Ère industrielle

Latest Era: Ère de l'information


Pacte d'espionnage

Les espions effectuant l'opération choisie profitent virtuellement de 2 niveaux de plus.

L'opération choisie n'est pas disponible.

Earliest Era: Ère de la Renaissance

Latest Era: Ère atomique


Traité mondial sur l'énergie

Interdit la Production de bâtiments de ce type.

Coût en Production -50% pour ce type de bâtiment.

Earliest Era: Ère de la Renaissance


Organisation culturelle

Tourisme doublé pour les chefs-d'œuvre de ce type.

Aucun Tourisme pour les chefs-d'œuvre de ce type.

Earliest Era: Ère industrielle


Doctrine de luxe

Tous les exemplaires de la ressource de luxe ciblée octroient de l'Activité.

La ressource de luxe choisie n'octroie pas d'Activité.


Sociétés de mercenaires

Le coût en Production ou d'achat des unités militaires est augmenté de 50% jusqu'au prochain Congrès mondial.

Le coût en Production ou d'achat des unités militaires est réduit de 25% jusqu'au prochain Congrès mondial.


Traité de migration

Croissance 20% plus rapide, mais loyauté par tour -5 dans les villes du joueur ciblé.

Loyauté par tour +5, mais croissance -20% dans les villes du joueur ciblé.

Earliest Era: Ère médiévale


Conseil militaire

Puissance de combat +3 pour les unités de la classe de promotion choisie.

Puissance de combat -3 pour les unités de la classe de promotion choisie.

Latest Era: Ère atomique


Mécénat

Double les points générés pour ce type de Personnage illustre.

Aucun point généré pour ce type de Personnage illustre.

Latest Era: Ère moderne


Programme de travaux publics

Production +100% pour ce projet.

Production -50% pour ce projet.

Earliest Era: Ère industrielle

Latest Era: Ère de l'information


Souveraineté

Type de rendement de la Cité-état +100% pour les routes commerciales à destination de ce type de Cité-état.

Les Cités-états de ce type n'octroient pas leur bonus de suzerain exclusif.

Latest Era: Ère moderne


Doctrine sur le commerce

Or +4 pour les routes commerciales internationales vers le joueur ciblé. Routes commerciales +1 pour le joueur ciblé.

Or -4 pour les routes commerciales internationales depuis ou vers le joueur ciblé.


Traité d'urbanisation

Production +100% pour les bâtiments de ce quartier.

Aucun bâtiment ne peut être construit dans ce quartier.

Latest Era: Ère moderne


Idéologie mondiale

Ce type de gouvernement reçoit un emplacement de doctrine Joker.

Ce type de gouvernement perd un emplacement de doctrine Joker.

Earliest Era: Ère médiévale


Religion mondiale

Puissance de combat religieuse +10 pour toutes les unités religieuses de cette religion.

Tous les joueurs peuvent condamner les unités de la religion choisie, ce qui leur octroie 25  faveurs diplomatiques.

Latest Era: Ère industrielle
