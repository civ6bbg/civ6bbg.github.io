Civ VI Base Game World Wonders Bonuses Description


Ère antique

Etemenanki


+2 Science

Science +2 et Production +1 pour toutes les cases de marais dans votre empire. Science +1 et  Production +1 pour toutes les cases de plaines inondables de votre ville.

Doit être construite sur une case de plaines inondables ou de marais.

Coût de production = 110


Grand bain


+3 Habitations

+1 Activités

Les cases de plaines inondables qui longent la rivière où est situé le Grand bain sont désormais immunisées contre les dégâts causés par les crues (mais les bonus de nourriture et de production consécutifs sont réduits de moitié). Foi +1 pour les cases de plaines inondables de la ville chaque fois que les dégâts d'une crue sont limités.

Doit être construit sur une case de plaine inondable.

Coût de production = 90


Jardins suspendus


+2 Habitations

La durée de vie de tous les héros augmente de 10%. Augmente la croissance de toutes les villes de 15%. Doivent être construits près d'une rivière.

Coût de production = 90


Oracle


+1 Culture

+1 Foi

Le mécénat des personnages illustres nécessite 25 % de foi en moins. Les quartiers de la ville font gagner 2 points de personnage illustre de même type qu'eux (par exemple, des points d' écrivain illustre pour la place du théâtre). Doit être construit sur une colline.

Coût de production = 145


Pyramides


+2 Culture

Octroie un bâtisseur gratuit. Tous les bâtisseurs peuvent construire un aménagement supplémentaire. Doivent être construites dans un désert (plaines inondables de désert incluses) sans colline.

Coût de production = 110


Stonehenge


+2 Foi

Octroie un prophète illustre gratuit. Les prophètes illustres peuvent fonder une religion à Stonehenge plutôt que dans un lieu saint. Doit être adjacent à une ressource de pierre et construit sur un terrain plat.

Coût de production = 90

Coût de production = 90
