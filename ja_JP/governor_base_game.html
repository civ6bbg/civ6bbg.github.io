Civ VI Base Game Governors and their Promotions Description


ビクター


方形堡

都市の守備隊の 戦闘力+5。3ターンで完成する。


守備隊指揮官

都市の領土で防衛にあたるユニットの 戦闘力+5。9タイル以内にある自文明の他都市に対し、ターン毎の自文明への忠誠心+4。


防衛兵站

都市が攻囲されなくなる。この都市で戦略資源を蓄積すると、毎ターンさらに+1。


銃眼

ターン毎に都市がおこなえる 長距離攻撃の回数+1。この都市で訓練された軍事ユニットは、最初から無償のレベルアップを1つ持っている (無償のレベルアップをまだ持っていない場合)。


防空構想

航空機やICBMによる攻撃からの防衛時、都市内の対空支援ユニットの 戦闘力+25。


軍拡競争支持者

都市のすべての核兵器計画の 生産力+30%。


アマニ


伝令

都市国家へ配属可能。 代表団2つに相当する。


公使

9タイル以内にある自文明が保有していない都市について、ターン毎に忠誠心-2。


潤沢

都市国家に着任した場合、そこの高級資源と同じものが得られる。


現地情報提供者

この都市で活動する敵スパイは、レベルが通常より3つ下がる。


外国人投資家

都市国家に置くと、その戦略資源が得られる。宗主国である場合は、得られる戦略資源が2倍になる。


人形遣い

都市国家に着任した場合、そこにいる自文明の 代表団の数が2倍になる。


モクシャ


司教

都市が隣接都市に与える宗教的圧力+100%。都市の専門区域1つにつき 信仰力+2。


大審問官

都市内での神学戦争で 宗教戦闘力+10。


癒しの手

この都市のタイルにいる 総督の全ユニットが、1ターンで完全回復する。


神の砦

都市が、 総督のプレイヤーが創始したのではない宗教による圧力を受けない。建設が完了すると建設コストの25%に相当する 信仰力を得る。


守護聖人

都市で訓練された使徒と武僧が、最初のレベルアップ時に追加のレベルアップ1つを得る。


聖なる建築者

都市が 信仰力を使って区域を購入できる。


マグナス


先駆者

都市での収穫や特徴の削除による産出物+50%。


余剰管理

都市の 成長+20%。自分の 交易路がここで終わる場合、開始地点の都市に 食料+2。


備蓄

都市の 人口を消費せずに開拓者を訓練できる。


実業家

石炭火力発電所、石油火力発電所、原子力発電所の各資源によってもたらされる 電力+1、 生産力+2。


闇商人

ユニットが消費する戦略資源が80%減る。


垂直統合

都市が、直近だけでなく、近隣に存在する、地域ボーナスを持つすべての工業地帯の建造物から 生産力を得る。


梁


ギルドマスター

この都市で訓練されたすべての労働者の労働力+1。


都市計画委員

都市の区域の建設に対する 生産力+20%。


養殖漁業

養殖場を建設できる。 食料+1、 住宅+0.5。海洋資源に隣接している場合、さらに 食料+1。梁が都市にいる場合、 生産力+1。


強化素材

この都市の施設、建造物、区域は環境効果によるダメージを受けない。


上水道

この都市にある近郊部や用水路1つにつき 住宅+2。この都市にある運河やダム区域1つにつき 快適性+1。


公園と娯楽

この都市に都市公園を建設できる。 文化力+1、アピール+2。水域に接している場合 快適性+1。梁が都市にいる場合、都市公園は 文化力+3をもたらす。


ピンガラ


司書

都市が産出する 科学力と 文化力+15%。


鑑定家

この都市の 市民1人につき毎ターン 文化力+1。


研究員

この都市の 市民1人につき毎ターン 科学力+1。


助成者

都市のターン毎の 偉人ポイント+100%。


学芸員

着任した都市の傑作 (芸術、音楽、書物) による 観光力+100%。


宇宙開発構想

着任した都市のすべての宇宙開発プログラムに対する 生産力+30%。


レイナ


土地取得

都市の新タイル取得が早まる。この都市を通る外国の 交易路1つにつき ゴールド+3。


港長

都市の商業ハブと港の隣接ボーナスが2倍になる。


森林管理

未開発のタイルごとに ゴールド+2。この都市で未開発のタイルに隣接しているタイルのアピール+1。


収税吏

毎ターン、都市の 人口1につき ゴールド+2。


建築請負人

都市が ゴールドで区域を購入できるようになる。


再生可能エネルギーへの補助金

この都市にあるすべての洋上風力発電所、太陽光発電所、風力発電所、地熱発電所、水力発電ダムの 電力と ゴールド+2。


イブラヒム


パシャ

この都市では、すべての軍事ユニットに対する 生産力+20%。


鷹匠頭

都市の領土で戦うすべての味方ユニットの 戦闘力+5。


セラスケル

防御に適した区域を攻撃する際、都心から10タイル以内にいる全ユニットの 戦闘力+10。


ハーッス・オダ・バシ

同盟国の 首都に着任した場合、その文明とのターン毎の同盟ポイント+0.5。


カプ・アガー

他文明の 首都に着任した場合、その文明のあなたに対する 不平を1ターンにつき1低下させる。


大宰相

他文明の 首都に着任した場合、その文明の都市が忠誠心へ圧力をかけてこなくなる。
