Civ VI BBG 5.7 Weltkongress


Waffenkontrolle

Die Massenvernichtungswaffen aller Spieler werden auf den Wert des Zielspielers gesetzt.

Der Zielspieler verliert alle Massenvernichtungswaffen.

Earliest Era: Moderne


Grenzkontrollabkommen

Neue von diesem Spieler gebaute Bezirke lösen einen Kulturschock aus.

Die Grenzen dieses Spielers wachsen nicht durch Kultur.

Latest Era: Moderne


Abholzungsvertrag

Verbietet das Entfernen von Geländearten des gewählten Typs.

Die Rodung von Geländearten des gewählten Typs bringt Gold-Ertrag in Höhe der Produktion und Nahrung.

Earliest Era: Industriezeitalter

Latest Era: Informationszeitalter


Spionagepakt

Spione, die die ausgewählte Operation ausführen, funktionieren 2 Stufen höher.

Die gewählte Operation ist nicht verfügbar.

Earliest Era: Renaissance

Latest Era: Atomzeitalter


Globaler Energievertrag

Die Produktion dieses Gebäudetyps ächten.

50 % Nachlass bei der Produktion von Gebäuden dieses Typs.

Earliest Era: Renaissance


Erbschaftsorganisation

Tourismus durch Große Werke dieses Typs wird verdoppelt.

Kein Tourismus durch Große Werke dieses Typs.

Earliest Era: Industriezeitalter


Luxuspolitik

Duplikate des gewählten Luxusguts gewähren Annehmlichkeiten.

Das gewählte Luxusgut gewährt keine Annehmlichkeiten.


Söldnerkompanien

Die Produktion oder der Kauf von Militäreinheiten mit der gewählten Währung kostet bis zum nächsten Weltkongress +100 %.

Die Produktion oder der Kauf von Militäreinheiten mit der gewählten Währung kostet bis zum nächsten Weltkongress -50 %.


Migrationsabkommen

+20 % schnelleres Bevölkerungswachstum, aber -5 Loyalität pro Runde in Städten dieses Spielers.

+5 Loyalität pro Runde, aber -20 % Bevölkerungswachstum in Städten dieses Spielers.

Earliest Era: Mittelalter


Militärberater

Einheiten der gewählten Beförderungsklasse erhalten +5 Kampfstärke.

Einheiten der gewählten Beförderungsklasse verlieren 5 Kampfstärke.

Latest Era: Atomzeitalter


Patronat

Verdiene doppelte Punkte bei Großen Persönlichkeiten dieser Klasse.

Es werden keine Punkte bei Großen Persönlichkeiten dieser Klasse verdient.

Latest Era: Moderne


Öffentliches Bauvorhabenprogramm

+100 % Produktion für dieses Projekt.

-50 % Produktion für dieses Projekt.

Earliest Era: Industriezeitalter

Latest Era: Informationszeitalter


Souveränität

+100 % vom Ertragstyp des Stadtstaats, wenn Ihr Handelswege zu einem Stadtstaat dieses Typs schickt.

Stadtstaaten dieses Typs gewähren ihren einzigartigen Suzerän-Bonus nicht.

Latest Era: Moderne


Handelspolitik

Internationale Handelswege, die an den gewählten Spieler geschickt werden, bringen dem Absender +4 Gold. Der gewählte Spieler erhält +1 Handelsweg-Kapazität.

Internationale Handelswege, die zu dem gewählten Spieler führen oder von dem gewählten Spieler gesendet werden, bringen -4 Gold (kann negativ sein).


Stadtentwicklungsabkommen

+100 % Produktion für Gebäude in diesem Bezirk.

In diesem Bezirk können keine Gebäude gebaut werden.

Latest Era: Moderne


Weltideologie

Diese Regierungsform erhält einen Joker-Politikplatz.

Diese Regierungsform verliert einen Joker-Politikplatz.

Earliest Era: Mittelalter


Weltreligion

+10 religiöse Stärke für alle religiösen Einheiten dieser Religion im theologischen Kampf.

Alle Spieler können Einheiten der gewählten Religion verurteilen. Dies gewährt 25 diplomatische Gunst.

Latest Era: Industriezeitalter
