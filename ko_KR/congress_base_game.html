Civ VI Base Game 세계 의회


군비 규제

모든 플레이어의 대량살상무기가 대상 플레이어와 동일하게 설정됨

대상 플레이어가 모든 대량살상무기를 잃음

Earliest Era: 원자 시대


국경 통제 조약

이 플레이어가 건설한 새 특수지구가 문화 폭탄으로 기능

이 플레이어의 국경이 문화를 통해 성장하지 않음

Latest Era: 현대 시대


산림 파괴 협약

선택한 유형 특성을 베거나 제거하는 행위 금지

선택한 유형 특성을 제거하면 생산력 및 식량과 동일한 양의 금 생산

Earliest Era: 원자 시대

Latest Era: 정보 시대


첩보 협정

스파이가, 선택한 작전 기능을 2단계 높게 실행합니다.

선택한 작전을 사용할 수 없습니다.

Earliest Era: 산업 시대

Latest Era: 원자 시대


국제 에너지 협약

이 유형의 건물 생산을 금지

이 유형의 건물 생산에 50% 할인

Earliest Era: 현대 시대


유산 기구

이 유형의 걸작에서 두 배의 관광이 제공됨

이 유형의 걸작에서 관광이 제공되지 않음

Earliest Era: 현대 시대


사치 정책

선택한 사치 자원을 여러 개 보유하고 있어도 쾌적도 보너스 부여

선택한 사치 자원이 쾌적도 0 부여


용병 회사

선택한 화폐 유형을 사용하여 군사 유닛을 생산하거나 구매하는 것은 차기 세계 의회 때까지 비용이 +100%입니다.

선택한 화폐 유형을 사용하여 군사 유닛을 생산하거나 구매하는 것은 차기 세계 의회 때까지 비용이 -50%입니다.


이주 조약

이 플레이어의 도시에 인구 성장 속도 +20% 및 턴당 충성심 -5

이 플레이어의 도시에 턴당 충성심 +5 및 인구 성장 -20%

Earliest Era: 산업 시대


군사 고문

선택한 진급 부류 유닛에 전투력 +5

선택한 진급 부류 유닛에 전투력 -5

Latest Era: 원자 시대


후원

이 클래스의 위인에 대해 두 배의 점수를 획득함

이 클래스의 위인에 대해 점수를 획득하지 않음

Latest Era: 현대 시대


홍보 활동

선택한 플레이어는 100% 더 많은 적대감을 생성하며, 다른 플레이어는 이 플레이어에 대해 100% 더 많은 적대감을 생성합니다.

선택한 플레이어는 50% 더 적은 적대감을 생성하며, 다른 플레이어는 이 플레이어에 대해 50% 더 적은 적대감을 생성합니다.

Latest Era: 원자 시대


공공 사업 프로그램

이 프로젝트에 대한 생산력 +100%

이 프로젝트에 대한 생산력 -50%

Earliest Era: 원자 시대

Latest Era: 정보 시대


자주권

교역로를 이 유형의 도시 국가로 보낼 경우 도시 국가의 생산량 유형 +100%

이 도시 국가 유형은 특유 종주국 보너스를 제공하지 않음

Latest Era: 현대 시대


무역 정책

선택한 플레이어에게 보낸 교역로에서 출발 문명에게 금 +4, 선택한 플레이어는 교역로 최대치 +1

다른 문명과 선택한 플레이어 사이의 모든 국제 교역로가 취소되고, 새로운 문명과의 교역 금지


도심 개발 조약

이 특수지구의 건물에 대한 생산력 +100%

이 특수지구에는 건물을 건설할 수 없음

Latest Era: 현대 시대


세계 이데올로기

이 정부 유형이 와일드카드 정책 슬롯을 획득

이 정부 유형이 와일드카드 정책 슬롯을 잃음

Earliest Era: 현대 시대


세계 종교

선택한 종교의 모든 종교 유닛에 전투력 또는 신앙력 +10 부여

모든 플레이어가 선택한 종교의 유닛을 비판할 수 있으며 이 경우 외교적 환심 25 부여

Latest Era: 산업 시대
