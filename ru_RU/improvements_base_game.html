    Civ VI Base Game Улучшения


    Алькасар

    Добыча: 25

    Модификатор обороны: 4

    Размещенный здесь юнит получил бонус для укрепления.

    +50% к науке от привлекательности клетки.

    Престиж для ближайших клеток: 0

    +2 Культура

    100% туризма от {3_YieldName[2]} (требуется Авиация Технология)

    Создается: Строители

    Valid Terrains: Пустыня, Пустыня (Холмы), Луг, Луг (Холмы), Равнина, Равнина (Холмы), Снег, Снег (Холмы), Тундра, Тундра (Холмы)

    ------------------------------

    Позволяет строителям строить алькасары.

    +2 ед. культуры. Дает науку в количестве, равном 50% престижа клетки. Находящийся здесь юнит получает +4 к обороноспособности и автоматически получает укрепление на 2 хода. Дает туризм после исследования авиации. Нельзя построить рядом с другим алькасаром.

    Позволяет строителям строить асьенды (уникально для Великой Колумбии).

    +2 ед. золота, +1 ед. производства и +0,5 ед. жилья. +1 ед. пищи за каждые две соседние плантации (повышается за каждую плантацию со «Стандартизацией»). Плантации и асьенды получают +1 ед. производства за каждые две соседние плантации (повышается за каждую асьенду с «Быстрым развертыванием»). Можно строить только на равнинах и лугах (с холмами или без них).

    Горный тоннель

    Престиж для ближайших клеток: 0

    Создается: Военный инженер

    Valid Terrains: Пустыня (Горы), Луг (Горы), Равнина (Горы), Снег (Горы), Тундра (Горы)

    Открыто: Химия Технология

    ------------------------------

    Позволяет военным инженерам строить горный тоннель.

    Действует как портал, ведущий сквозь горную цепь: юниты проходят через него и выходят уже из другого портала, расходуя на это 2 очка перемещения. Торговые пути, проходящие по этой постройке, получают больше золота от районов в пункте назначения. Можно построить только рядом с клеткой гор. Нельзя разграбить или убрать.
